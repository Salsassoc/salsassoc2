Salsassoc
============

Salsassoc is a web application to manage an association.

- Github: https://github.com/Salsassoc/salsassoc2
- Bugs reporting: https://github.com/Salsassoc/salsassoc2/issues

Features
--------
- Users managment
- Membership managment
- Accounting managment
